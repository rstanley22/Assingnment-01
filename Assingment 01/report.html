Also that I feel that it looks better and is more visualy pleaseing to the eye. The header fonts I decided to go with more of a gaming font, as I play computer games frequently and this reflects my peronality.

Grids

During the module we were shown grids, this is an incredable tool to use as this allows you to break down the page into smaller sections or boxes. However there are two methods for setting grids the first being, Grid template areas where you can name the areas within the grid to locate the placement for your tags. The second is grid row and column where you state which row and column you want the tag to be in. I prefer using the second method as this gives you more control and freedom to edit your webpage.
